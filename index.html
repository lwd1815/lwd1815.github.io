半亩绿林


平时看好多大神都有属于自己的专属博客,每每想起,总是羡慕的不行,所以今天我也决定建立一个属于自己的博客,虽然不是大神,但追赶大神的脚步不能停不是.

1 需要准备的条件

专属域名+github pages+hexo框架

github pages和域名不会操作的可以看鸿祥大神的这篇博客

http://blog.csdn.net/lmj623565791/article/details/51319147


2 hexo踩的坑

需要安装nodejs和git
安装好nodejs和git后,用管理员身份打开git bash

先安装淘宝镜像cnpm

npm install -g cnpm –registry=https://registry.npm.taobao.org

然后安装hexo

cnpm install -g hexo-cli

安装好hexo后

1,在本地新建一个文件夹blog

2,利用管理员权限打开git bash

3,cd进入到此文件夹下面

4初始化hexo

hexo init

5,配置_config.yml

6因为3.0版本的需要安装deployer的server

以管理员身份打开git bash,进入到blog目录下

cnpm install hexo-deployer-git –save


7安装后依次执行下列命令


删除已经生成的静态页面

hexo clean

生成博客的静态页面

hexo generate

将生成的静态页面上传到github

hexo deploy

3 替换hexo的主题

https://hexo.io/themes/

选择一个主题,复制主题github的地址

进入到blog目录,在themes目录下,克隆

例如

https:github.com/iHogRend/legend themes/legend


然后在hexo的_config.yml文件中把配置主题

如theme:legend

然后根据主题中的使用规则自行配置

配置好后继续执行

hexo c

hexo g

hexo d


创建新的page

hexo new page xxx

然后在layout中创建相应的ejs文件,配置参考现有的
